Problema espacial

Lixo orbital ameaça serviços essenciais na Terra

A órbita terrestre está cada vez mais congestionada por satélites desativados, estágios de foguetes e fragmentos de colisões. Esses detritos viajam em alta velocidade e podem danificar satélites ativos responsáveis por comunicação, navegação, monitoramento climático, sensoriamento remoto e resposta a desastres.

Solução Hermes

Rede de serviços orbitais sob demanda

1

Módulo de Serviço Orbital Universal

Executa inspeção, aproximação, captura, taxiamento e desorbitagem controlada de detritos.

2

Estação de Abastecimento Orbital

Funciona como base logística permanente para reabastecimento, acoplamento e troca de dados.

3

Módulo de Transporte de Combustível

Leva combustível da Terra até a estação e permite reutilização em futuras missões.

Arquitetura AWS

Deploy simples, seguro e automatizado

GitHub GitHub Actions IAM Role OIDC S3 privado CloudFront URL pública HTTPS

O projeto foi pensado para publicação em S3 com bucket privado, distribuição via CloudFront, deploy automatizado por GitHub Actions, segurança com IAM Role, Parameter Store e KMS, além de monitoramento com AWS Budget e CloudWatch.
